# Success Metrics & KPI Tracking

## North Star Metric
- **Total Audited Savings**: The sum of all "Potential Savings" identified across all completed audits.

## Primary KPIs
1. **Conversion Rate (Form Start -> Completion)**: Goal > 40%.
2. **Lead Conversion Rate (Report View -> Email Capture)**: Goal > 15%.
3. **Share Rate (Report URL Copied)**: Goal > 5%.

## Quality Metrics
- **Average Savings Per Audit**: Target > $500/mo.
- **AI Summary Rating**: (Future feature) User feedback on the summary quality.

## Technical Metrics
- **Report Generation Time**: Target < 3 seconds.
- **Supabase Write Success Rate**: Target > 99.9%.
